DANCE ELEVATED

I was recently contacted by Chelsie Hightower, who asked me to update and change her website for her company's dance camps. It's a WordPress website that was built using the theme "The7". I generally prefer to code websites, but I also build WordPress websites when a client either prefers it, or already has one that they need to make changes to. This one was pretty fun to work on, and I was able to do a bit of custom CSS on top of the WP theme to get a little more control of things. The website is used to display information about her dance camps, and it also gives people the ability to register for her camps. Not only is Chelsie an phenomenal dancer, but she's an amazing person as well.

NFL ARRESTS

This is a website that I built using HTML/CSS, SCSS, Gulp, Angular, and ES6, and I utilized Angular's UI Router for the page views. The website makes calls to a public API that provides information about all arrests of NFL players dating back to January of the year 2001. I created some CSS animations for each page, as well. It started as a way to just practice making API calls in Angular, and then I decided to make it into an actual website! Click the project image to view the website!



WORKOUT TRACKER

This is a website that I built for tracking and logging workouts. You can log runs, bike rides, and gym workouts, so that you can set goals and keep track of your progress. The front-end is built with HTML/CSS, Angular, Gulp, SASS, ES6, with Auth0 for log-in and the back-end uses Node.js, Express, and SQL. The website was ready to be hosted, when the motherboard on my old computer decided it was done functioning. I have since gotten it transferred onto a new computer, but the Postgres database on the new computer isn't playing nice, so I'm in the process of setting up a new database program for the back-end before hosting it. Check back soon to view the website!
